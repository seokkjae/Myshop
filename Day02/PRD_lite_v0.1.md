## 1. 개요 (Overview)

서울 망원동(마포구) 골목 끝의 동네 카페 **「모닝브루」**. 출근 전 동네를 지나는 직장인과 거주민에게, **이른 아침 시간대에 갓 내린 커피와 가벼운 아침거리를 빠르게 제공**하고 단골 관계로 묶어내는 것을 목표로 한다.

- **한 줄 정의**: "망원동에서 가장 일찍 켜지는 불빛 — 당신의 아침을 여는 한 잔."
- **사업 단계**: 초기 컨셉/MVP 검증 단계
- **이 문서의 범위**: 제품(카페 경험·메뉴·운영)의 v0.1 방향성 정의. 인테리어·재무 상세는 별도 문서로 분리.

---

## 2. 문제 정의 (Problem)

출근 전 짧은 시간(7~9시) 동안 동네에서 만족스러운 커피 경험을 얻기 어렵다. 핵심 페인은 [[시장조사_타깃페인]]에서 도출:

1. 출근 피크 시간대 대기 줄 → 시간 부족
2. 이른 아침에 문 연 동네 카페가 드묾
3. 테이크아웃·빠른 픽업에 최적화되지 않음
4. 커피와 함께 먹을 간단한 아침거리 부족
5. 단골이 되어도 매번 주문을 처음부터 설명 (단골 인식·적립 부재)

---

## 3. 타깃 고객 (Target Users)

- **1차 타깃**: 망원동을 지나 출근하는 20~40대 직장인 (시간 압박 + 테이크아웃 중심)
- **2차 타깃**: 이른 아침 산책·운동 후 들르는 동네 거주민
- **공통 맥락**: "출근 전 5분", 가격보다 **시간·신선함·익숙함**을 중시

---

## 4. 솔루션 개요 (Solution)

가격 경쟁(저가 프랜차이즈·편의점)으로는 이기지 않는다. **이른 아침 시간대 선점 + 동네 단골 관계 + 아침 루틴 경험**이라는 비가격 축에 집중한다. (근거: [[경쟁자_비교분석]])

| 페인 | 솔루션 방향 |
|---|---|
| 대기 줄 | 사전주문·픽업존 등 빠른 동선 |
| 늦은 오픈 | 경쟁사보다 이른 오픈 시간 |
| 테이크아웃 비최적 | 테이크아웃 전용 픽업 동선·홀더 |
| 아침거리 부족 | 커피+간단 푸드(베이글·스콘 등) 세트 |
| 단골 미인식 | '나의 주문' 저장·간편 적립 |

---

## 5. 핵심 기능 / 제공 가치 (Key Features — v0.1 MVP)

| 우선순위 | 기능 | 설명 |
|---|---|---|
| P0 | **얼리 오픈 운영** | 경쟁 동네 카페보다 이른 시간 개점, 출근 시간대 선점 |
| P0 | **빠른 픽업 동선** | 테이크아웃 전용 픽업존·간편 결제로 대기 최소화 |
| P0 | **시그니처 모닝 메뉴** | 갓 내린 커피 + 간단 아침거리 세트(가격 합리화) |
| P1 | **단골 간편 적립·주문 저장** | 종이 쿠폰 또는 간단한 멤버십으로 단골 인식 |
| P1 | **사전주문(모바일/전화)** | 픽업 시간 지정, 줄 서지 않는 경험 |
| P2 | **동네 관계 콘텐츠** | 원두 스토리, 단골 이름 기억 등 비가격 차별화 |

## 6. 성공 지표 (Success Metrics)

- **북극성 지표**: 주중 오전(7~9시) 시간대 객수 / 재방문 단골 비율
- 보조 지표: 객단가, 테이크아웃 평균 대기 시간, 사전주문 비중, 단골 재방문 주기
- *측정값 목표치는 실제 데이터 확보 후 설정 (확인 필요)*

## 7. 범위 (Scope)

**포함 (v0.1)**
- 오전 출근 시간대 중심 운영·메뉴·픽업 동선
- 간단한 단골 적립 체계

**미포함 (Out of Scope, 차기)**
- 배달 앱 입점 / 자체 앱 개발
- 좌석 중심 '머무는 카페' 확장, 야간 운영
- 로스팅·원두 리테일 판매
